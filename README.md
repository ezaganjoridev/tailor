# Yorkdale Tailoring Website

A basic tailoring website for Yorkdale Tailoring Services located near Yorkdale Mall, serving the Greater Toronto Area.

## Features

- Professional tailoring services showcase
- Service pricing information
- Customer reviews with carousel
- Contact form (via Formspree)
- Review submission form
- Mobile-responsive design
- SEO optimized

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Formspree (form handling)

## Local Development

Run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`
